# Flat Tracker — парсер аренды квартир с Циана

Парсит 1-комнатные квартиры в аренду в Москве до 60 000 ₽/мес через неофициальный API Циана и отображает результаты в виде красивой веб-страницы.

## Стек

- **Python 3.9+** + `requests` — парсинг
- **HTML + CSS + JS** (без фреймворков) — интерфейс

## Быстрый старт

```bash
# 1. Установить зависимости
pip3 install requests

# 2. Запустить парсер (соберёт ~140 объявлений, сохранит в data.json)
python3 parser.py

# 3. Запустить локальный сервер
python3 -m http.server 8000
```

Открыть в браузере: [http://localhost:8000](http://localhost:8000)

## Возможности интерфейса

- Карточки с фото, ценой, метро, площадью и этажом
- Переключение в режим таблицы
- Слайдер фильтрации по цене
- Сортировка по цене и площади
- Прямые ссылки на объявления

## Структура проекта

```
flat-tracker/
├── parser.py    # парсер Циана
├── index.html   # веб-интерфейс
├── data.json    # результаты парсинга (генерируется, не в git)
└── .gitignore
```

## Примечание

`data.json` исключён из git — он генерируется локально при каждом запуске `parser.py`. Cian иногда добавляет капчу; если парсер вернул 0 объявлений, попробуйте запустить снова через несколько минут.
